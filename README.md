📚 Reading Checklist App
A simple Reading Checklist Web Application built with Spring Boot, Thymeleaf, HTML, and CSS. Users can add books to their reading list and mark them as completed.

✨ Features
➕ Add books to your reading checklist

📖 View all books in your list

✅ Mark books as completed/uncompleted

🎨 Clean and intuitive user interface

🛠️ Tech Stack
Java - Programming language

Spring Boot - Backend framework

Spring MVC - Web layer

Thymeleaf - Template engine

HTML/CSS - Frontend styling

Maven - Dependency management

🚀 Quick Start
Clone the repository

bash
git clone https://github.com/yourusername/reading-checklist.git
Navigate to project directory

bash
cd reading-checklist
Run the application

bash
mvn spring-boot:run
Open in browser

text
http://localhost:8080
